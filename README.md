# Learning & Certification Tracker

A personal, dark-themed web app to showcase my learning journey and online certifications, presented as a professional portfolio. Built as a static site and hosted free on GitHub Pages, with edits that go live everywhere in real time.

**Live site:** https://mimi0320mje.github.io/learning-tracker/

## About

This site is my portfolio of projects and certifications. Visitors can browse:

- 🗂️ **Projects** — things I've built, with links, dates, and the tech used
- 🎓 **Certifications** — courses and credentials I've completed, including an **AI** category
- 🌌 A calming galaxy dark theme

The content updates instantly for every visitor whenever I add something new.

## Features

- Galaxy dark theme with a clean, readable layout
- Tabs for **Overview**, **Projects**, and **Certifications**
- A private edit mode (only I can make changes; everyone else can view)
- A **Contact me** button to get in touch

## Tech

Plain HTML, CSS, and JavaScript — no build step. Hosted on GitHub Pages with a free cloud backend so updates appear live for everyone.

## Files

- `index.html` — the entire app
- `data.json` — fallback seed data
